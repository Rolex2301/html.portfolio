My projects:

Auto and Taxi Manager:

A java program with a dedicated frontend created in NetBeans which collects
details of various customers and taxi drivers and stores them in a manipulatable
backend DBMS table

Industrial Water monitoring system for farmers:

Consists of various sensors such as a temperature sensor, pH sensor, turbidity sensor,
and conductivity sensor connected to an Arduino Uno board along with the LCDs for
displaying the details of the water sample from which the farmers can know if they are
fit to use for agriculture

Image steganalysis using discrete cosine transform:

This project attempts to perform image steganalysis, that is to detect whether
a given image has any encrypted information in it. Uses neural network model
which takes DCTR of the image as features.
